## Мета
Навчитися створювати SSH ключі, працювати з Git, GitHub та документацією.
## Контрольні запитання
1. Для чого використовують SSH?  
   Для захищеного підключення до GitHub/GitLab.

2. Що таке репозиторій?  
   Сховище проєкту з історією змін.

3. Що таке гілка (branch)?  
   Незалежна лінія розвитку проєкту.
